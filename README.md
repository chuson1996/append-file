#Append file

---

A node module to easily insert content from a file to another. Check out the example below.

---

```
<!-- dest.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title></title>
    <style>
        /*<sou.css>*/
        /*</sou.css>*/
    </style>
</head>
<body>
</body>
</html>
```
